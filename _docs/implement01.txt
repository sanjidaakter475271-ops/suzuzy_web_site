- Immutable audit logs with trigger protection
- Feature flag system for gradual rollout
Migration 5: RLS Policies & Functions
- Row Level Security policies for all tables
- Helper functions (generate_order_number, generate_invoice_number)
- Triggers (updated_at timestamp, audit_log protection)
